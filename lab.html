Observability & Defensive Architecture

MONITORING

Prometheus & Grafana Dashboard

Full-stack observability for all infrastructure components. Prometheus handles time-series data collection while Grafana provides a "Single Pane of Glass" for real-time performance and health visualization.

SIEM / EDR

Wazuh Security Monitoring

Centralized Security Information and Event Management (SIEM). Deployment of Wazuh agents across all VMs and containers for real-time log analysis, file integrity monitoring, and vulnerability detection.

ACTIVE DEFENSE

CrowdSec & Brute Force Mitigation

Multi-layered IP reputation and brute-force defense. Utilizing CrowdSec agents with localized bouncers to automatically ban malicious actors at the firewall level across the entire infrastructure.

SECURE INGRESS

Identity-Aware Access (Cloudflare)

Accessing localized resources without open firewall ports. Implementing Cloudflare Tunnels to provide encrypted, authenticated ingress to specialized services behind a secure edge.

ORCHESTRATION

Virtualization Stack

Proxmox VE Cluster managing segregated LXC and KVM environments. Automated snapshotting and Infrastructure-as-Code testing for situational awareness tools like TAK Server.
